# Documentation (v0.4.0)

`microlens_defects` is a classical schlieren / phase-shift microlens inspection
toolkit with two pillars: **microstructure morphology** (five-step
phase-shifting) and **threshold-based defect detection**.

- **Getting started**: see the repository [`README.md`](../README.md) for
  install and CLI examples (`detect`, `phase5`).
- **Architecture & pipeline**: [architecture.md](architecture.md) — components,
  inputs/outputs, configuration.
- **Python API**: [api_guide.md](api_guide.md) — module-level usage.
- **Troubleshooting**: [troubleshooting.md](troubleshooting.md) — common
  install/data/runtime issues.
- **Changelog**: [../CHANGELOG.md](../CHANGELOG.md).

> Learning-based detection is developed in a separate project; this package is
> the classical morphology / defect front end.
