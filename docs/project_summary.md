# 项目背景信息总结

## 1. 项目目标

本项目旨在开发一个面向实验室研究人员的集成化软件系统，用于对离焦微结构镜片的图像数据进行管理、分析和缺陷检测。系统将整合经典图像处理算法与现代人工智能技术，实现从数据导入、初步缺陷定位，到数据标注、AI模型训练、智能识别及结果统计分析的全流程功能，为镜片质量评估和工艺改进提供数据支持。

## 2. 系统架构

系统采用轻量化、模块化的单机架构，优先考虑灵活性和易于研究探索。

*   **数据层 (Data Layer):** 使用 **SQLite** 数据库存储图像元数据、标注信息及检测结果。文件系统用于存储原始及处理后的图像文件。数据库表结构设计灵活，以适应研究过程中缺陷类型的动态调整。
*   **核心功能库 (Core Library):** 一个统一的 Python 功能库，包含数据处理、相位计算、经典缺陷检测、AI模型（训练/推理）等所有核心算法模块，供上层应用直接调用。
*   **用户表示层 (Presentation Layer):** 采用“工具箱”策略，提供多种交互界面：
    *   **研究与开发:** **Jupyter Notebook** 用于算法验证和探索性分析。
    *   **数据标注:** 本地部署的 **Label Studio** 用于高效、灵活地制作数据集。
    *   **日常使用:** 一个统一的桌面应用（推荐使用 **Streamlit** 快速开发）集成所有功能，提供给实验人员进行一站式操作。

## 3. 核心功能

*   **数据导入与管理:** 自动化扫描、整理图像文件，并将其元数据存入数据库。
*   **图像分析:** 实现五步相移法波前计算和基于传统算法的初步缺陷检测。
*   **数据标注:** 支持对缺陷进行分类和精确分割（分割掩码）标注，构建AI训练数据集。
*   **智能检测:** 支持AI模型的训练、评估，并使用训练好的模型对新图像进行缺陷智能识别。
*   **可视化与统计:** 提供多结果对比视图（原始图、相位图、算法/模型结果）和缺陷数据统计看板。

## 4. 技术栈

*   **后端/核心:** Python 3.x
*   **数据库:** SQLite
*   **核心库:** NumPy, Pandas, Pillow, Scipy, PyTorch/TensorFlow
*   **标注工具:** Label Studio
*   **研究界面:** JupyterLab
*   **用户应用:** Streamlit 或 PyQt/PySide
